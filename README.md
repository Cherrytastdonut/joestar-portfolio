## 🚀 Overview (제왕의 선언)
단순히 정보를 나열하는 포트폴리오는 버려라! 
이 프로젝트는 **브라우저의 한계를 초월한 텍스트 렌더링 최적화**와 **인터랙티브 UX**의 결합을 통해, 개발자 이강건이 문제를 대하는 '구조적 태도'를 시각적으로 증명합니다.

## 💥 Stand Abilities (핵심 기술)

### 1. Pretext 실시간 리플로우 엔진
- **@chenglou/pretext**를 활용하여 DOM의 성능 한계를 극복.
- 수천 자의 텍스트가 120fps의 부드러움으로 이미지의 경계를 따라 **파문처럼** 재배치됩니다.

### 2. 구조적 문제 해결 (The Tracker)
- Android/Kotlin 기반의 탄탄한 기본기 위에 Java, Spring, Flutter를 얹어 실제 서비스의 결과를 도출합니다.
- 오류를 만났을 때 **"포기하지 않고 원인을 끝까지 추적"**하여 구조 자체를 개선하는 제왕의 집념이 코드에 녹아있습니다.

### 3. 클라우드 아키텍처
- **AWS S3 + CloudFront**를 통해 정적 자산을 전 세계에 빛의 속도로 전달합니다.

## 🛠️ Tech Stack
- **Core**: Next.js (App Router), React
- **Typography Engine**: @chenglou/pretext
- **Animation**: Framer Motion
- **Styling**: Tailwind CSS
- **Infrastructure**: AWS (S3, CloudFront)
